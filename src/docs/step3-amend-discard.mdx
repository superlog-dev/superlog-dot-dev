# Amend and Discard

Superlog

---

## Amend button

You can use "Amend" button to amend any selected changes into current commit. This is equivalent to:

1. git amend
2. rebase all the children commit to the newly amended commit

## Discard button

You can use "Discard" button to discard any selected changes.
