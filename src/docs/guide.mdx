# Superlog Essentials

<Notice title="What you'll learn" type="tldr">

- End-to-end stacked pull request workflow using Superlog, without any git command involved.

</Notice>

---

Superlog is a git GUI app that optimized for stacked pull request workflow.

Stacked pull request workflow enable people to stay unblock and ship code faster, but has only been enjoyed by those who are expert in git.

Superlog want to make this workflow accessible to every engineer, especially for those who are not comfortable with git.
Life is too short to master those obsecured git commands.
